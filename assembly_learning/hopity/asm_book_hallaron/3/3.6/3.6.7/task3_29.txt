Оператор continue в языке C заставляет программу перейти в конец текущей итерации
цикла. При использовании оператора continue правило преобразования цикла for в цикл
while требует некоторой доработки. Например, рассмотрим следующий код на C:

/* Пример цикла for с оператором continue */
/* Вычисляет сумму нечетных чисел между 0 и 9 */
long sum = 0;
long i;
for (i = 0; i < 10; i++)
{
    if (i & 1)
    continue;
    sum += i;
}

1. Что получится, если просто применить правило преобразования цикла for в цикл while,
описанное выше? Какую ошибку допустит такой код?
2. Как можно заменить оператор continue оператором goto, чтобы гарантировать правильное преобразование цикла for в цикл while?

__________________________________________________________________________________

1. continue не обновит счётчик
2. выделить меткой "counter" обновление счётчика и заменить continue на goto counter;
